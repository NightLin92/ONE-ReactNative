## Development Workflow

### Step One

```
git clone https://github.com/kenvies/ONE-ReactNative.git
```
### Step Two

```
cd ONE-ReactNative && npm install
```
### Step Three

```
react-native run-ios/run-adnroid(建议用xcode或者android studio去编译你的原生项目)
```
